# Web-Scrapping-using-Python

•	Developed a Python-based web scraping tool using requests and BeautifulSoup to extract book data (title, price, and ratings) from a multi-page e-commerce website.

•	Implemented dynamic user input filtering, allowing selection of specific rating levels (1–5 stars) and automated traversal across all paginated pages.

•	Stored cleaned and filtered data efficiently into a structured text file for analysis, demonstrating practical data extraction and parsing skills in Python.
